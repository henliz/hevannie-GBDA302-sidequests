## Description
A Journey-inspired splash screen built in p5.js to confirm GitHub Pages setup. A pastel blue and lavender sky gradient fades to seafoam at the horizon. Cherry blossom petals drift downward with a gentle sway, landing into five layers of animated ocean waves at the bottom. The name and course tag sit centred with a soft pink glow behind them.

Bonus: Splash screen designed with a Journey-inspired colour palette and mood - soft, meditative, and atmospheric.

---
## Setup and Interaction Instructions
No setup required - opens directly in the browser via GitHub Pages.
No interaction needed, the animation plays on its own.
Resizing the window rescales the canvas automatically.
